# Discount Microservice

## Stack

- Discount
    - NodeJs
    - NestJs (IoC)
    - Typescript
    - MongoDB


## Build and Run

```
npm run build
```

```
npm run start
```

## Testing

```
npm run test
```

You can use BloomRPC and do a request to *0.0.0.0:50051*
```
{
  "userId": "1",
  "productId": "1"
}
```
